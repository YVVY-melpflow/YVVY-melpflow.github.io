# くどうれいん『スノードームの捨てかた』

- 読了日：2025/09/20
- 点数　：82 点

物語というのは，ある登場人物が何かしらのイベントに出会って，その状況がどう推移するかを描いたものだと思うが，この短編集ではどの人間のどの瞬間を切り取ってんねんというような話が並んでいる．  
しかし描写にはリアリティがあり，現代のどこかにはこういう人間がいてこういうイベントが起こっている感じがする．  
『鰐を送ったら「会いたい」だって，わかれよ』 この理不尽な思考を描写しているのはすごいなと思った．  
あと「湯気」怖すぎる．
